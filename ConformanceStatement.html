OsiriX DICOM Conformance Statement

Last update of this document: 11/12/06


Introduction

This section is an abbreviated DICOM conformance statement for OsiriX.

It specifies the compliance of OsiriX to file reading/writing, file displaying and file transferring support to the dicom standard.

Classical values

Description

8, 12, 16
